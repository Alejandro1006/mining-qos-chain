# Seguridad y Privacidad
- **Sin PII** en este repositorio. Documentos con datos personales quedan en repos con permisos (OpenProject/Nextcloud).
- On-chain guardaremos **hashes (keccak-256)** e identificadores no legibles.
- Secretos/tokens: no se versionan en Git.
- Rama `main` protegida: PR obligatorio y 1 aprobación (PO).
